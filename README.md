# Weather App (Fetch API)

A beginner-friendly weather app built as a capstone project.  
It uses the Fetch API to get weather data and display it in the browser.

## Features
- Search for a city’s weather
- Show temperature and conditions
- Clean and simple design

## How to Use
1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Enter a city name to see the weather.
